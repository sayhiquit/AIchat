# AI人际管家

AI人际管家是一个本地优先的人际关系处理助手原型，用来管理人物关系、沉淀人物画像、辅助回复与汇报、导入聊天记录进行画像蒸馏，并保留离线使用和数据迁移能力。

当前项目包含网页版原型和 Electron 桌面版壳。网页版用于快速测试功能，桌面版用于真实使用和后续分发。

## 当前能力

- 人际关系地图：按同事、朋友、亲人、客户、上一家公司、同学等分类查看人物。
- 分类和人物管理：支持新增、编辑、删除分类与人物。
- 人物背景：可记录关系分量、亲近程度、联系频率、现实背景。
- 对话助手：支持“对方所说”和“我方述说”两种模式，生成回复选项和策略分析。
- 画像蒸馏：可粘贴或导入聊天记录，更新说话风格、做事风格、好感触发、避雷点。
- 记忆管理：模拟对话默认不写入长期记忆，点击新增记忆并保存后才会沉淀。
- 数据迁移：支持导出和导入本地记忆 JSON。
- AI 接入：设置页支持 DeepSeek / OpenAI 兼容接口配置和连接测试。
- 桌面版：Electron 加载同一套页面，用户数据不会随安装包覆盖。

## 本地运行

```powershell
npm.cmd install
npm.cmd run validate
npm.cmd run desktop
```

如果只看网页版，也可以直接打开 `index.html`。

## 打包 Windows 桌面版

```powershell
npm.cmd run dist:win
```

安装包输出到 `release/`。打包前请先更新 `package.json` 版本号，并运行：

```powershell
npm.cmd run validate
```

## 数据和隐私

当前数据默认保存在本地浏览器或桌面应用数据目录中。源码和安装包只包含默认模板，不会把你本地真实人物、API Key、聊天记录打包给别人。

桌面版用户数据通常位于：

```text
%APPDATA%\AI人际管家
```

正式上线云同步前，建议继续使用离线模式和本地导出备份。

## 开发约定

- 网页版是测试版，先在网页版迭代功能。
- 桌面版是真实使用版，阶段性同步网页版能力。
- 每次同步桌面版或打包前，必须运行 `npm.cmd run validate`。
- 打包新版本时必须更新版本号。
- 不提交 `node_modules/`、`release/`、构建临时目录、真实用户数据和 API Key。
