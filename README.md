# AI人际管家

AI人际管家是一个本地优先的人际关系处理助手，用来管理人物关系、沉淀人物画像、辅助回复、策略咨询、日程维护和聊天记录蒸馏。当前主线已经切到桌面软件版，`index.html`、`app.js`、`styles.css` 仍是 Electron 加载的界面源码，但后续默认按软件版真实使用、验证和打包。

## 当前能力

- 人际关系地图：按同事、朋友、亲戚、客户、同学等分类查看人物，支持分支展开、收缩、聚焦和小屏侧栏收起。
- 关系管理：分类和人物支持增删改查，人物可补充背景、重要程度、亲近程度、联系频率和现实关系说明。
- 人物详情：展示说话风格、做事风格、好感触发、避雷点、回复结构、关系策略和历史记忆。
- 对话助手：接近 QQ / 微信的聊天结构，支持对方开场、我方开场、推荐选项、自定义回复、AI 预测对方回复和修改真实回复后继续生成建议。
- 画像蒸馏：可手动粘贴片段，也可导入聊天记录文件，先预览统计和样例，确认后才写入画像。
- 模拟与真实记忆分离：普通模拟不写入真实历史，点击新增记忆并保存后才进入长期记忆。
- 日程表：支持自动建议和手动日程，手动事项可完成、延期、忽略或删除，并可绑定人物。
- AI 设置：支持 DeepSeek 预设、OpenAI 兼容接口、自定义 Base URL、模型名、API Key 和连接测试。
- 数据迁移：支持普通 JSON 导出导入，也支持带密码的本地加密备份。
- 诊断模块：设置页可查看版本、人物数量、记忆数量、AI 状态、存储位置、最近备份时间等。

## 本地运行

```powershell
npm.cmd install
npm.cmd run validate
npm.cmd run desktop
```

## 打包 Windows 软件

```powershell
npm.cmd run validate
npm.cmd run dist:win
```

安装包输出到 `release/`。每次软件版更新必须同步更新 `package.json`、`package-lock.json` 和界面里的版本号，并在打包前运行验证。

## 数据与隐私

源码和安装包只包含默认模板和程序文件，不会把本机真实人物、聊天记录、API Key 打包给别人。桌面版用户数据通常位于：

```text
%APPDATA%\AI人际管家
```

正式上线账号、云同步和后端代理前，建议继续使用离线模式、本地导出和加密备份。

## 开发约定

- 软件版是真实使用主线，网页版不再作为独立产品维护。
- 修改功能后先跑 `npm.cmd run validate`，再打包软件。
- `release/`、`node_modules/`、真实用户数据、真实 API Key 不提交到 Git。
- release 目录只保留最新版本完整产物和历史版本 `.exe` 安装包。
- 后续上线服务器时，需要重新评估后端代理、登录同步、自动更新、聊天软件辅助和移动端实时分析。
