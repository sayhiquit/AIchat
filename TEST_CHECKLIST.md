# 回归测试清单

每次优化、打包桌面版或准备分发前，按这份清单跑一遍。

## 1. 基础验证

- 运行 `npm.cmd run validate`，确认全部 OK。
- 运行 `npm.cmd run desktop`，确认软件首页无白屏。
- 切换浅色 / 深色模式，确认主要页面文字可读。
- 缩小窗口到最小宽度，确认布局不严重遮挡。
- 关闭重开软件，确认本地数据仍在。

## 2. 人际网

- 点击分类节点，确认地图移动、分支展开和退场收缩正常。
- 展开一个分类后切换另一个分类，确认旧分支不会残留。
- 点击人物节点，确认人物聚焦、其他节点渐隐、右侧详情出现。
- 点击头像或返回，确认小球平滑恢复，不突然缩小。
- 小屏下确认左侧工作区可隐藏和展开。

## 3. 分类和人物 CRUD

- 新增、编辑、删除分类。
- 删除分类后，确认人物被迁移到可用分类。
- 新增人物时点击取消，不应保存空人物。
- 编辑人物背景、重要程度、亲近程度、联系频率，确认详情展示正确。
- 删除人物时红色提示清晰，二次确认正常，删除后不残留选中状态。
- 管理页搜索、重要程度筛选和排序正常。

## 4. 人物详情与蒸馏

- 手动输入聊天片段并蒸馏，确认画像和记忆更新。
- 导入 TXT / CSV / JSON / Markdown / HTML，确认出现预览。
- 确认导入后才蒸馏，取消导入不改变画像。
- 导入 BAK / BAT 等异常文件时，提示清楚且不白屏。
- 编辑和删除历史记忆后刷新，确认结果仍在。

## 5. 对话助手

- 对方开场：输入对方的话并发送，对方气泡先出现，再出现我方推荐气泡。
- 点击 A/B/C 后，内容进入底部输入框。
- 自定义发送后，推荐气泡消失，我方气泡展示实际发送内容。
- 我方开场：我方消息先出现，对方显示正在输入中，再生成对方预测回复和我方建议。
- 修改 AI 预测的对方回复后，确认能重新生成我方建议。
- 点击新增记忆但未保存时，切换页面应提示未保存。
- 未保存的模拟对话刷新后清空，已保存真实历史保留。

## 6. AI 设置

- DeepSeek 预设可填入 Base URL 和模型名。
- 未填写 API Key 时测试连接应提示缺少配置。
- 填写 Key 后测试连接应显示成功或明确错误。
- 关闭在线 AI 后，对话和蒸馏应回退到本地规则模拟。
- 在线 AI 请求失败时，页面不卡死，有回退提示。

## 7. 日程

- 打开日程页，确认时间表、月历、周计划正常。
- 新增手动日程，确认出现在时间表里。
- 绑定人物后，点击查看人物能跳到详情。
- 完成、延期、忽略、删除手动日程，刷新后状态正确。

## 8. 数据迁移

- 普通导出 JSON，确认文件能下载。
- 加密导出，确认需要设置密码并能下载。
- 导入普通 JSON，确认人物、分类、记忆和日程恢复。
- 导入加密 JSON，确认密码错误会失败，密码正确能恢复。
- 设置页确认当前存储位置和最近备份时间展示合理。

## 9. 诊断和发布

- 设置页诊断模块可刷新。
- 版本号与 `package.json` 一致。
- `git status --short` 只包含预期改动。
- 不提交真实 API Key、真实人物库、聊天记录、`node_modules/` 和 `release/`。
- 打包后确认 `release/` 只保留最新完整产物和历史 `.exe` 安装包。
